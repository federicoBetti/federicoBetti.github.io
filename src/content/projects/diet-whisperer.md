## Overview

Diet Whisperer is an AI-driven diet planning platform that generates personalized meal plans from user preferences, supports real-time editing, and publishes plans as calendar events.

## Links

- GitHub: [federicoBetti/diet-whisperer](https://github.com/federicoBetti/diet-whisperer)

## Highlights

- **Agentic planning workflow**: Meal plan generation using a structured LangGraph-based agent.
- **Calendar integration**: Export/subscribe via ICS so plans appear in Google/Outlook/Apple Calendar.
- **Full-stack delivery**: FastAPI backend + database persistence, modern React/Next.js UI, and container-ready deployment.

## Tech stack

- **Backend**: FastAPI, PostgreSQL (prod) / SQLite (dev)
- **Frontend**: Next.js, React, Tailwind CSS
- **AI**: LangGraph + LLM tooling
- **Deployment**: Docker, Heroku
