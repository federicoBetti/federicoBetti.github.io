## Overview

Menu Muse helps users make better choices at restaurants by analyzing menu content (including multi-language menus) and returning structured, actionable suggestions.

## Links

- Backend (API): [federicoBetti/menu-muse-python](https://github.com/federicoBetti/menu-muse-python)
- Frontend (Web): [federicoBetti/menu-muse-eats-you](https://github.com/federicoBetti/menu-muse-eats-you)

## Highlights

- **Backend-first design**: An API service focused on reliability, clear endpoints, and deployability.
- **Practical integrations**: Image storage/retrieval via Supabase and deployment-ready containerization.
- **Delivery engineering**: CI/CD and production deployment workflows (Heroku).

## Tech stack (high level)

- **Backend**: FastAPI (Python)
- **Frontend**: Web UI (separate repo)
- **Storage**: Supabase
- **Deployment**: Docker, Heroku
